# mc-rk3588

* 电源采用12V、24V均可
* 支持亮钻的D3588 Linux/Android 系统，整体可以启动，需要微调一下外设USB、网卡。
* 带mstata接口、4G接口、5G接口、HDMI、6个USB
* 属于研发板子，部分带有飞线
* 内存8G，emmc 64G
